# "Daato (Working title)"

A 2D platformer and shooting game built with Unity.  
This project is a game featuring simple movement, shooting mechanics, and score tracking

The game is in a Work In Progress state. It is not possible to play it yet.

---

## About the Game

The core mechanic allows the player to shoot darts, enabling them to shot at different types of target, in order to gain scores and achievements.
Achievements are required to progress through the game, you must defeat various challenges in the overworld or in special challenge rooms to obtain them.

---

## Features

- 2D character movement and jumping
- Precise shooting with physics
- Object destruction
- Fast travel mechanic
- Small timed challenges  
- Graphic design approach to environment art / "Faked" glitched world
- Interactable NPC with dialogue

## Controls

- **A / D** – Move  
- **Space** – Jump  
- **Left Mouse Click** – Aim
- **Left Mouse Click Release** – Shoot
- **E** - Interact
